Установка Сервера

Сервер выложен на github: https://github.com/AlexeyGrishin/smart-dog

Инструкция по установке:

  1. Установите последний Node.js
  2. Убедитесь, что путь к бинарникам прописался в PATH - команды 'node' и 'npm' должны выполняться в консоли без проблем
  3. Скачайте сервер одним из следующих способов:
    • Установите git и выполните git clone git://github.com/AlexeyGrishin/smart-dog.git в нужной вам директории
    • Скачайте его себе в виде ZIP архива
  4. Перейдите в папку с сервером
  5. npm install
  6. ./server.bat или ./server.sh
  7. Откройте в браузере http://localhost:3000
  8. Все готово! Можете запустить тестовых ботов и начать писать своего: 
    cd clients\js-client
start player1.bat
start player2.bat
    или 
    ./clients/js-client/player1.sh &
./clients/js-client/player2.sh &

Полезная информация

  • Сервер читает конфигурацию из ./games/smart-dog/server_config.json . Этот файл не хранится в репозитории и создается при первом запуске сервера из шаблона ./games/smart-dog/server_config.source.json.
  • Карты лежат в ./maps
  • По-умолчанию сервер хранит все данные в памяти, которые теряются при остановке сервера. Чтобы хранить данные в базе CouchDB:
    1. Установите CouchDB
    2. в server_config.json поменяйте строчку 
         "storage": "mem""couch"
    3. Перезапустите сервер. Теперь все игры хранятся в базе между перезапусками сервера

    При желании вы можете реализовать поддержку своей базы. Для этого нужно создать класс, который реализует те же методы, что и у классов MemoryStorage и CouchDBStorage, и инициализировать его в smart_dog_server.js из настроек.
