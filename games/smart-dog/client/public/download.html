Установка Сервера

Сервер выложен на github: https://github.com/AlexeyGrishin/smart-dog

Инструкция по установке:

  1. Установите последний Node.js
  2. Убедитесь, что путь к бинарникам прописался в PATH - команды 'node' и 'npm' должны выполняться в консоли без проблем
  3. Скачайте сервер одним из следующих способов:
    • Установите git и выполните git clone git://github.com/AlexeyGrishin/smart-dog.git в нужной вам директории
    • Скачайте его себе в виде ZIP архива
  4. Перейдите в папку с сервером
  5. npm install
  6. ./server.bat или ./server.sh
  7. Откройте в браузере http://localhost:3000
  8. Все готово! Можете запустить тестовых ботов и начать писать своего: 
    cd clients\js-client
start player1.bat
start player2.bat
    или 
    ./clients/js-client/player1.sh &
./clients/js-client/player2.sh &

Полезная информация

  • Конфигурация сервера лежит в ./games/smart-dog/server_config.json
  • Карты лежат в ./maps
